============ FORMULAS DE TEMPERTURA =================

🌡️ Celsius ↔ Fahrenheit

C -> F 

F = (C * 9/5)+32

F -> C

C = (F - 32)*5/9
=========================
🌡️ Celsius ↔ Kelvin

C -> K

K = C + 273.15

C = K -  273.15
==========================
🌡️ Fahrenheit ↔ Kelvin

F -> K 

K = (F-32)*5/9 + 273.15

F = (K - 273.15)*9/5 + 32
===========================
